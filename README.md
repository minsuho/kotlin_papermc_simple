# Kotlin PaperMC Simple Template

## 사용된 라이브러리
- [Kommand](https://github.com/monun/kommand)

## Configuration
gradle.properties로 버전과 그룹, 메인 클래스를 지정 할 수 있습니다.

## How to build
IntelliJ IDEA 기준으로 우측 Gradle 탭에서 `해당 프로젝트 -> Tasks -> build -> jar`를 더블클릭하여 빌드합니다.
